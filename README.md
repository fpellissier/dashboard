# Dashboard OOPRO

1 - Introduction :

This is a node.js server based app, and vuejs client based to display a dashboard.

2 - Installation :

To install all dependencies, simply run :

	$ npm install

3 - Running test :

DEV Environnement :

	$ npm test

4 - Running app :

	$ npm start 

and then load the page on your browser at addresses : 

	https://localhost:3003/client
